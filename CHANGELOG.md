# 更新日志

本文件记录对用户可感知的主要功能更新与关键修复。GitHub Releases 会自动读取对应版本的内容作为发布说明。

## [0.3.2] - 2026-07-11

### 语音输入

- 新增长按全局快捷键语音输入，并保证按下与松开事件有序处理
- 快捷键触发方式改为下拉选择，配置更直观

### 实时识别

- 音量持续低于阈值时自动断开 ASR 流，减少无效连接
- 修复 ASR 异步启动后可能残留连接的问题
- 修复无会话时误显示加载状态、实时识别误显示音频波纹的问题

## [0.3.1] - 2026-07-11

### 字幕与 OBS

- 实时字幕顶部集中提供桌面与 OBS 输出目标切换，默认输出到桌面；OBS 未准备就绪时会自动回落至桌面悬浮窗
- 输出到 OBS 时保留全部字幕样式设置，并明确引导在 OBS 画布中调整字幕位置，避免误以为位置设置丢失
- 优化字幕预览入口与状态反馈，便于在配置样式时快速确认效果

## [0.3.0] - 2026-07-08

### 功能更新

- 实时字幕新增翻译能力，接入 Qwen-MT 字幕翻译模型，悬浮窗可同步显示原文与译文
- 听写支持通过全局快捷键一键清除输入内容

### 结构优化

- 供应商与识别模型改为注册表驱动，新增供应商或模型不再需要改动多处硬编码分支
- 拆分听写、实时字幕、字幕转写、文稿对齐等模块中的大文件，按职责重新划分目录结构

### 细节修复与优化

- 修复译文悬浮窗行为与原文不一致的问题，并按标点分句提升翻译速度
- 修复托盘打开主窗口后尺寸持续变大，以及关闭按钮 hover 状态残留的问题
- 修复设置页 API Key 显示/隐藏图标语义反了、明文状态下点击即重新打码的问题
- 修复 Fun-ASR-Flash 过期回声句导致字幕重复的问题
- 修复听写下拉列表中非实时模型丢失「（非实时）」后缀及排序错乱的问题
- 优化字幕预览开关位置、样式设置栏对齐，以及翻译模型命名规范

## [0.2.2] - 2026-07-06

首个正式版本。「说吧！」是一款桌面端语音输入与听写工具，目前提供：

- 语音输入：按快捷键即可将语音转换为文字，支持多种识别模型
- 实时字幕：悬浮窗展示语音识别的实时字幕
- 字幕转写：导入本地音视频文件，生成纯文本或字幕，内置字幕编辑器
- 文稿对齐：将现成文稿与识别时间轴自动对齐，支持导出 SRT
- 热词与本地文本处理，提升识别准确率
- 录音降噪、响度与均衡器调节

### 优化

- 优化外部链接打开方式，提升稳定性

## [0.2.0] - 2026-07-05

### 功能更新

- 新增「字幕转写」，可导入本地音视频文件生成纯文本或字幕，并内置字幕编辑器。
- 新增「文稿对齐」，支持把现成文稿与识别时间轴自动对齐，并导出 SRT。
- 语音输入新增非实时识别模型支持，并补充更多阿里云百炼模型选项。
- 设置页新增多模型效果对比，便于快速比较不同模型识别结果。
- 桌面端界面整体升级，统一到新的设计令牌体系与深蓝黑视觉风格。
- 录音调校能力增强，支持降噪、响度与均衡器试听调节。

### 关键修复与优化

- 修复短音频识别 `400 Download failed`，并在同步短音频路径自动压缩为 Opus / MP3，降低体积超限概率。
- 修复 Fun-ASR-Flash 在短音频场景下的响应解析错误、无字幕与流式重复字幕问题。
- 优化字幕编辑器的时间轴交互、波形缩放、播放头同步与字幕间隙合并，降低暂停恢复卡顿和字幕闪烁。
- 优化本地文本处理连续标点合并规则、实时字幕阴影裁切、主窗口启动居中和快捷键按钮对齐等体验细节。
