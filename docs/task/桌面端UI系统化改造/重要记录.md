## 记录 001：参考图优先于参考 HTML

- 日期：2026-07-04
- 状态：已确定
- 背景：用户提供了目标设计图和 `docs/others/stitch_speak_bar_ui_reconstruction` 中的参考代码。参考代码是静态 HTML，包含 CDN Tailwind、外部图标和静态交互，不完全适配当前 React/Tauri 项目。
- 最终决定：视觉方向以参考图为准，参考 HTML 只用于理解布局关系、控件密度和局部样式，不直接复制。
- 选择原因：当前项目已有业务逻辑、组件、store、Tauri 交互和桌面端窗口能力，直接复制静态代码会破坏可维护性和功能链路。
- 影响任务：1.1、1.2、2.1、2.2、3.1、3.2

- 日期：2026-07-04
- 状态：已确定
- 背景：本任务目标是 UI 系统化改造，不是重做语音输入、字幕、录音识别、设置或后端能力。
