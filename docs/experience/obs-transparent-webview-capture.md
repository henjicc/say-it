# OBS 无法捕获透明字幕窗口

## 触发条件

Windows 10 上，OBS 使用窗口采集读取 Tauri/WebView2 的透明字幕悬浮窗时，预览可能只显示黑帧。单纯关闭 WebView2 GPU 渲染并不能稳定修复这个问题。

## 处理方式

保留正常的透明字幕窗口用于桌面显示；另外创建一个非透明的 `说吧！OBS 字幕采集` 镜像窗口，使用纯绿色背景输出同一份字幕内容，并将其放在屏幕外。OBS 采集镜像窗口后，通过“色度键”滤镜移除绿色背景。

## 验证

重启桌面应用后，打开实时字幕，在 OBS 的窗口采集中选择“说吧！OBS 字幕采集”，确认字幕文本可见且会实时更新；随后添加“色度键”滤镜并选择绿色作为关键色。
